# Lab 03 — MPI Collectives

## คำอธิบาย

แบบฝึกหัดนี้สาธิตการใช้ MPI_Allreduce ซึ่งเป็นปฏิบัติการรวมค่าแบบกลุ่ม (collective operation)
ที่รวมข้อมูลจากทุกโพรเซสแล้วกระจายผลลัพธ์กลับไปยังทุกโพรเซส
พร้อมวัดเวลาแฝงสำหรับขนาดข้อมูลต่าง ๆ เพื่อทำความเข้าใจประสิทธิภาพ

## ไฟล์ในแบบฝึกหัด

| ไฟล์ | คำอธิบาย |
|------|----------|
| `allreduce_test.c` | ทดสอบ MPI_Allreduce พร้อมวัดเวลาแฝงหลายขนาดข้อมูล |

## ข้อกำหนดเบื้องต้น

- ติดตั้ง MPI (OpenMPI หรือ MPICH)
- คอมไพเลอร์ C ที่มาพร้อม MPI (mpicc)

```bash
# ติดตั้ง OpenMPI (Ubuntu/Debian)
sudo apt install openmpi-bin libopenmpi-dev

# หรือติดตั้ง MPICH
# sudo apt install mpich libmpich-dev
```

## วิธีคอมไพล์

```bash
make            # คอมไพล์ทุกโปรแกรม
make clean      # ลบไฟล์ที่คอมไพล์แล้ว
```

## วิธีรัน

```bash
# รันด้วย 4 โพรเซสบนเครื่องเดียว
mpirun -np 4 ./allreduce_test

# รันด้วย 8 โพรเซสบนหลายเครื่อง
mpirun -np 8 --hostfile hosts.txt ./allreduce_test
```

## สิ่งที่ได้เรียนรู้

1. การใช้ MPI_Init / MPI_Finalize เริ่มต้นและจบการทำงาน MPI
2. การใช้ MPI_Allreduce สำหรับรวมค่าแบบกลุ่ม
3. เทคนิคการวัดประสิทธิภาพ: รอบอุ่นเครื่อง, Barrier, วัดค่าเฉลี่ย
4. ความสัมพันธ์ระหว่างขนาดข้อมูลกับเวลาแฝงและแบนด์วิดท์
