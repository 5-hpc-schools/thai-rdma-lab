# ห้องปฏิบัติการเขียนโปรแกรมอาร์ดีเอ็มเอ

โค้ดตัวอย่างและแบบฝึกหัดปฏิบัติสำหรับการแข่งขันเขียนโปรแกรมอาร์ดีเอ็มเอภูมิภาคเอเชียแปซิฟิก ครั้งที่ 12

## ข้อกำหนดเบื้องต้น

- ระบบลินุกซ์ที่มีอินฟินีแบนด์หรืออาร์ดีเอ็มเอบนอีเทอร์เน็ต
- ไดรเวอร์ OFED ของเมลลาน็อกซ์ (MOFED) หรือ rdma-core
- HPC-X หรือ OpenMPI ที่รองรับยูซีเอ็กซ์/ยูซีซี
- คอมไพเลอร์ GCC >= 8.0
- (เฉพาะ lab04) CUDA Toolkit >= 11.0 และเอ็นซีซีแอล

## การติดตั้ง

```bash
git clone <repository-url>
cd rdma-labs
source scripts/setup_env.sh
make all
```

## โครงสร้างห้องปฏิบัติการ

| ห้องปฏิบัติการ | บทที่ | หัวข้อ |
|---------------|-------|-------|
| lab01-verbs-basics | 3 | พื้นฐานเวิร์บส์: ส่ง/รับ เขียนอาร์ดีเอ็มเอ อ่านอาร์ดีเอ็มเอ |
| lab02-ucx-messaging | 4 | ส่วนต่อประสานยูซีเอ็กซ์: แท็ก ข้อความแอ็กทีฟ พุท/เก็ท |
| lab03-mpi-collectives | 5 | การสื่อสารเอ็มพีไอ: ปิงปอง ออลรีดิวซ์ ด้านเดียว |
| lab04-nccl-gpu | 6 | เอ็นซีซีแอลบนจีพียู: ออลรีดิวซ์ ขั้นตอนวิธีวงแหวน |
| lab05-ucc-collectives | 7 | กรอบงานยูซีซี: ออลรีดิวซ์ ออลทูออลด้านเดียว |
| lab06-competition-prep | 7+9 | เตรียมแข่งขัน: ออลรีดิวซ์ด้านเดียว เปรียบเทียบประสิทธิภาพ |

## การใช้งาน

```bash
# สร้างห้องปฏิบัติการเดียว
cd lab01-verbs-basics
make

# รันตัวอย่าง (ต้องมีอาร์ดีเอ็มเอ 2 โหนด)
mpirun -np 2 --map-by node ./send_recv

# รันชุดทดสอบประสิทธิภาพ
cd ../scripts
./run_benchmark.sh
```
